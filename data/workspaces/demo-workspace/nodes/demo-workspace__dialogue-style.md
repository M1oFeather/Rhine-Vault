# 对白风格初始约束

作为初始人物卡约束，白面鸮对话可采用冷静、资料化、带轻微系统提示感的表达方式。她可以关注生命体征、任务参数和风险提示，但不应被写成完全无情或脱离关怀的角色。该节点是创作辅助约束，需要在后续人工整理中继续校准。
