# Rhine-Vault Documentation

Rhine-Vault 是本地优先、数据主权优先的知识采集、审批、检索与 Context Bundle 基础设施。

当前公开文档分为两类：

- `architecture/`：冻结架构、领域模型、存储、检索、权限、恢复与 V1 范围。
- `implementation/`：当前 Full Implementation 状态、实施路线、Phase 1.5 / Phase 2 / Phase 3 / Phase 4 / Phase 5 / Phase 6 说明和项目风格标准。

本地 AI 协作材料、讨论记录、启动包和参考包放在 `docs/local/`、`docs/discussion/` 或 `docs/templates/`，默认不进入 MkDocs 站点。
